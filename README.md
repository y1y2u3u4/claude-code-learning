# Claude Code Learning

This is a learning repository for exploring Claude Code capabilities and GitHub integration.

## About

This repository serves as a practice space for:
- Understanding Claude Code workflow
- Testing GitHub issue and pull request interactions
- Learning collaborative development with AI assistance

## Getting Started

1. Clone this repository
2. Create issues to practice with Claude Code
3. Use `@claude` to trigger AI assistance on issues and pull requests

## Features

- Basic project structure
- GitHub integration examples
- Claude Code demonstration
